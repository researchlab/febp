无状态组件的优势
1.性能高, 因为它就是一个函数, 而普通组件是一个jsx中一个类， 其中会包含生命周期函数，还有render() ，比起无状态组件中只有render()函数 只要执行更多的生命周期函数;

2.无状态组件 一般可以用在UI组件定义;
